How to use /stations

POST -> http://206.189.26.231/stations with the json payload

EX: {"station_id": "70B3D549988A54E5_1", "new_state": "available"}

 

Possible outcomes:

- HTTP/1.1 200 OK - {"message": "success"}

- HTTP/1.1 403 FORBIDDEN - {"message": "too-many-requests"}

- HTTP/1.1 403 FORBIDDEN - {"message": "station-not-found"}

- HTTP/1.1 400 BAD REQUEST - {"message": "invalid state"}

- HTTP/1.1 400 BAD REQUEST - {"message": "invalid-json-format"}

- HTTP/1.1 400 BAD REQUEST - {"message": "station_id-or-new_state-not-present"}

 

You may experiment POST request on this website: https://apitester.com/
